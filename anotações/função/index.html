Função

É uma forma de organizarmos os dados e/ou compartilhar o código, na função existe os procedimento e as funções

procedimento

No procedimento não existe nenhum retorno

funções

Nas funções há o retorno

Regras

começar com minusculo, não pode começar com números Somente o caracter especial _ (underline) pode ser usado Colocar nomes significativos Se desejo usar uma variável de fora tem que ser encaminhada para dentro Usar nomes compreensiveis (sem 'X' ou 'Y' como nome)

argumentos(parametros)

É o local da função onde se faz a entrada de dados

return

É a saída de uma função (Procedimento não tem obrigatóriamente e funções tem obrigatoriamente)
